## Overview ##

This is my (Jeff McCune) personal Vim customizations.  To try out this
environment:

    cd ~
    git clone git://github.com/jeffmccune/jeff_vim .vim
    ln -s .vim/vimrc.vim .vimrc
    ln -s .vim/gvimrc.vim .gvimrc
    mkdir .vimswp
    vim
